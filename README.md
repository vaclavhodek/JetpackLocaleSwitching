# Switching locales with Jetpack Compose

This repository contains the source code for article [Switching locales with Jetpack Compose](https://localazy.com/blog/switching-locales-with-jetpack-compose).

---

## How To Use

1. Clone the repository.

2. Open it in Android Studio. As of now, the **Canary** version is necessary because of Jetpack Compose.

3. Run the app.
